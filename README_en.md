# SimpleGMAIL

![Project icon](icon.png)

[🇬🇧 EN](README_en.md) · [🇫🇷 FR](README.md)

✨ A Chrome extension that simplifies the Gmail interface for a cleaner, more focused experience.

## ✅ Features

- Hides the Gmail logo
- Reduces visual clutter (search bar, layout options)
- Centers email content and list
- Minimalist and transparent interface
- Lightweight CSS overlay, no JavaScript

## 🧠 Usage

1. Install the Chrome extension
2. Go to [Gmail](https://mail.google.com)
3. Enjoy a cleaner interface automatically

## ⚙️ Settings

No settings needed — the extension applies automatically.
