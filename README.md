# SimpleGMAIL

![Project icon](icon.png)

[🇫🇷 FR](README.md) · [🇬🇧 EN](README_en.md)

✨ Une extension Chrome qui simplifie l'interface de Gmail pour une expérience plus épurée et focalisée.

## ✅ Fonctionnalités

- Masque le logo Gmail
- Réduit l'encombrement visuel (barre de recherche, options de layout)
- Centre le contenu et la liste des mails
- Interface minimaliste et transparente
- Surcouche CSS légère, sans JavaScript

## 🧠 Utilisation

1. Installez l'extension Chrome
2. Allez sur [Gmail](https://mail.google.com)
3. Profitez d'une interface épurée automatiquement

## ⚙️ Réglages

Aucun réglage nécessaire — l'extension s'applique automatiquement.
